## Alfie Readme

This is a personal assistant App. The name Alfie is inspired by Batman's(Bruce Wayne) personal butler - Alfred. It has 4 main functionalities: 1. log and track all expenses, budget planning; 2. Calendar, review and track all your daily activities; 3. Set goals for each month, show achievements; 4. Get whatever questions answered by Alfie (ChatGPT API)
